Also, their `props` are passed as the first parameter to a function. Doesn't rerender if props/state hasn't changed. See: [Pure components](https://reactjs.org/docs/react-api.html#react.purecomponent) ### Component API undefined undefined undefined These methods and properties are available for `Component` instances.
