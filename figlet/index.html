    Figlet

    Dev Hints
    2019-05-28

    Usage

    Run figlet with -f <font> to use a font.

    Toilet fonts

    Figlet comes with support for Toilet fonts, which are in {share}/figlet. This is not the default font dir, so you’ll have to use -d <path> to access it.
